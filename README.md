# MVCHoldem
8th season of Telerik Academy - amateur ASP. Net MVC project



[![Coverage Status](https://coveralls.io/repos/github/gchankov/MVCHoldem/badge.svg?branch=)](https://coveralls.io/github/gchankov/MVCHoldem?branch=)

## Project Description
